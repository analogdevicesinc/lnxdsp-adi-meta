# ADSP Yocto meta layer

This repository contains the Yocto meta layer for ADI Digital Signal Processors
(ADSP).

See
[analogdevicesinc.github.io/lnxdsp-adi-meta](https://analogdevicesinc.github.io/lnxdsp-adi-meta/)
for documentation on how to get started and advanced configuration.

Please refer to the `LICENSE.md` file in this repository for more information
regarding licensing.

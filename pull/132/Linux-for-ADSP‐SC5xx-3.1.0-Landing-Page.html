Linux for ADSP-SC5xx Processors 3.1.0

Welcome to version 3.1.0 of Linux for ADSP-SC5xx. This collection of pages contains all you need to know in order to download, build, develop and install Linux, running on the ARM core of the ADSP-SC5xx family of processors.

Building of the Linux software requires a host PC running a recent flavour Linux. Analog Devices uses 64-bit Ubuntu 20.04 LTS for its building and testing of Linux for ADSP-SC5xx. If you are new to Linux development we strongly recommend using a host PC with this installation of Linux. If you are confident in resolving variant specific problems feel free to use your favourite distro.
